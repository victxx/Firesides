## Quick Start

1. Clone and install:
```bash
git clone https://github.com/your-username/firesides.git
cd firesides
npm install
```

2. Set up environment:
- Create `.env` file (see `.env.example`)
- Configure your keys

3. Start services:
```bash
# Terminal 1 - Backend
cd backend && npm start

# Terminal 2 - Frontend
cd frontend && npm start
```
